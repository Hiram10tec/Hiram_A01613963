    • Preguntas de laboratorio 1

    1. ¿Cuál es la diferencia entre Internet y la World Wide Web?

    Internet: Es la red global de redes de computadoras interconectadas. Permite la comunicación y el intercambio de información entre dispositivos ubicados en diferentes partes del mundo.

    World Wide Web (WWW):Es un servicio específico de internet que permite acceder y visualizar documentos multimedia (páginas web) a través de enlaces. La web es solo una parte de internet.

    2. ¿Cuáles son las partes de un URL?

    Protocolo: Especifica cómo se debe acceder al recurso (por ejemplo, "http" o "https").
    Dominio: La dirección única que identifica un sitio web.
    Ruta: La ubicación específica del recurso en el servidor.
    Parámetros: Información adicional, a menudo utilizada en la consulta de una base de datos.
    Fragmento: Una referencia a una sección específica dentro del recurso.

    3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    GET: Solicita datos del servidor.
    HEAD: Similar a GET, pero solo solicita encabezados, no el cuerpo del recurso.
    POST: Envía datos al servidor para ser procesados (utilizado comúnmente en formularios).
    PUT: Actualiza un recurso existente en el servidor.
    PATCH: Modifica parcialmente un recurso existente.
    DELETE: Elimina un recurso en el servidor.

    4. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML? ¿Por qué?

    Se utiliza POST para enviar datos al servidor de manera segura y confiable, como al enviar información de un formulario. Los datos no se muestran en la URL y pueden contener información sensible.
