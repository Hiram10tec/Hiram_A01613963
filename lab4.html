Preguntas de laboratorio 4
1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
Las diferencias entre Java y JavaScript son:
  • Aunque los dos son lenguajes de programación orientado a objetos, JavaScript esta basado en prototipos y multi-paradigma.
  • Java es un lenguaje de programación compilado, mientras que JavaScript es un lenguaje de programación interpretado.
Las semejanzas entre Java y JavaScript son:
  • Los dos sirven para desarrollar aplicaciones web o móviles.
  • Los dos son lenguajes de programación de alto nivel.
2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
  • getDate()
  • getDay()
  • getFullYear()
  • getHours()
  • getMilliseconds()
3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
  • concat()

4. ¿Cómo se declara una variable con alcance local dentro de una función?
Se puede declarar con let o const.
5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
Las variables globales pueden ser modificadas por cualquier función, lo que puede causar errores en el código.
